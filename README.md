# aviation-data-analysis
# ✈️ 항공편 지연 데이터 분석

## 프로젝트 소개
미국 항공편 데이터(2018~2023)를 분석하여
항공사별, 월별 지연 패턴을 파악한 프로젝트입니다.

## 사용 기술
- Python
- pandas
- matplotlib

## 데이터
- 출처: Kaggle - Flight Delay Dataset
- 기간: 2018 ~ 2023
- 규모: 약 3,000만 행, 29개 컬럼

## 분석 내용
1. 항공사별 평균 출발 지연시간 비교
2. 월별 평균 출발 지연시간 비교

## 주요 발견
- 6~7월 여름 성수기에 지연이 가장 많음
- 9월 비수기에 지연이 가장 적음
- JetBlue(B6)가 평균 지연시간 1위
